# Notes

## To run the Test:

- Go to `/vanilla_homework`
- `npm install`
- Alter the file `jest.config.js`
- `npm run test` / `npx jest`

## To link sources to an HTML:

- CSS: 

```html
<head>
    <!-- ... -->
    <link rel="stylesheet" href="style.css">
</head>
```

- JavaScript:

```html
<body>
    <!-- ... -->
    <script src="script.js"></script>
</body>
```

## Other

- `()`: parenthesis, `[]`: square bracket, `{}`: curly brace
